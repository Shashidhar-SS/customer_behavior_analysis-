🛍️ Customer Shopping Behavior Analysis

📌 Overview
An end-to-end Data Analytics project that analyzes customer shopping behavior using **Python, PostgreSQL, and Power BI**. The project focuses on data cleaning, SQL-based business analysis, and building an interactive dashboard to uncover actionable business insights.

🛠️ Tech Stack
- Python (Pandas, NumPy)
- PostgreSQL
- SQL
- Power BI
- Jupyter Notebook

 📊 Key Highlights
- Cleaned and transformed 3,900 customer records using Python.
- Handled missing values and performed feature engineering.
- Loaded cleaned data into PostgreSQL for SQL analysis.
- Solved 10 real-world business problems using SQL.
- Designed an interactive Power BI dashboard with KPI cards, slicers, and visualizations.

 📈 Dashboard Insights
- Customer Distribution
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group
- Subscription Analysis

💼 Skills Demonstrated
Python • SQL • PostgreSQL • Power BI • Data Cleaning • EDA • Feature Engineering • Data Visualization • Business Analysis
